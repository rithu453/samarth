How to Download the Extension

Follow these steps to install and use the Samarth extension effectively.

1. Download the Extension

Download the extension .zip file from the provided source.

2. Add the Extension Manually

  • Open your browser and go to: chrome://extensions/ (for Chrome) or edge://extensions/ (for Edge).
  • Enable Developer Mode (toggle switch in the top right corner).
  • Click on Load Unpacked.
  • Select the extension folder and upload it.

3. Activate the Extension

Once uploaded, click on the extension icon in the toolbar.

4. Navigate to the Required Website

Open the website where you want to use the extension.

5. Adjust Advanced Network Settings

  • Open DevTools (Ctrl + Shift + I or F12).
  • Go to the Network tab.
  • Click on Advanced Settings.
  • Select Accept to allow the extension to analyze the website.

6. Generate the Mind Map

  • Click on the extension icon again.
  • Select Generate Mind Map.
  • The extension will display all possible navigation paths in a mind map format.

7. Use the Chat Interface

You can ask any navigation-related questions in the provided chat interface.

The extension will guide you with relevant answers based on the mind map.
